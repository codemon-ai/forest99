# Phase 21: PWA 지원 (Progressive Web App)

## Overview
오프라인 플레이, 홈 화면 설치, 푸시 알림 지원.

## Files to Create

| 파일 | 설명 |
|------|------|
| `public/manifest.json` | PWA 매니페스트 |
| `public/sw.js` | 서비스 워커 |
| `public/icons/` | 앱 아이콘 (192x192, 512x512) |

## Files to Modify

| 파일 | 변경 |
|------|------|
| `index.html` | manifest 링크, 서비스 워커 등록 |
| `vite.config.js` | PWA 플러그인 설정 |

## Features

1. **오프라인 플레이**: 서비스 워커로 에셋 캐싱
2. **홈 화면 설치**: A2HS (Add to Home Screen) 지원
3. **앱 아이콘**: 다양한 크기의 아이콘 제공
4. **스플래시 스크린**: 앱 시작 화면

## Dependencies

```bash
npm install vite-plugin-pwa
```
