# 숲에서 보낸 99일 밤 - 개발 진행 현황

**프로젝트:** Three.js + React 기반 로우폴리 생존 게임  
**배포 URL:** https://forest99.vercel.app  
**최종 업데이트:** 2026-02-08 (Phase 25 완료)

---

## 기술 스택

- **프레임워크:** Vite + React 18
- **3D 렌더링:** Three.js + React Three Fiber + Drei
- **상태관리:** Zustand
- **오디오:** Howler.js
- **다국어:** i18next
- **스타일:** CSS

---

## 완료된 Phase (1-25)

### Phase 1-3: 프로젝트 설정 & 기본 월드 ✅

- Vite + React 프로젝트 초기화
- React Three Fiber 설정
- Perlin noise 기반 지형 생성
- 로우폴리 나무/바위 배치
- 플레이어 캐릭터 (WASD 이동, 마우스 시점)
- 3인칭 카메라 시스템

### Phase 4: 생존 시스템 ✅

- 체력(HP), 배고픔(Hunger), 정신력(Sanity)
- 베이스캠프 근처에서 정신력 회복
- HUD 표시

### Phase 5: 낮/밤 사이클 ✅

- 60초 주기 낮/밤 전환

- 동적 조명 효과

### Phase 6-7: 몬스터 시스템 ✅

- 주간: 뱀, 원숭이, 곰, 불개미

### Phase 8: 전투 시스템 ✅

### Phase 9: 인벤토리 & 크래프팅 ✅

### Phase 10: 이벤트 시스템 ✅
