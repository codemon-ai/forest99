# 🌲 숲에서 보낸 99일 밤 (99 Nights in the Forest)

> 99일간 숲에서 살아남아 숲의 수호자를 물리쳐라!

Three.js + React 기반 로우폴리 생존 게임입니다.

## 🎮 플레이하기

**[https://forest99.vercel.app](https://forest99.vercel.app)**

## 📖 스토리

당신은 미지의 숲에 갇혔습니다. 99일 동안 생존하여 숲의 수호자를 물리치고 탈출해야 합니다.
낮에는 자원을 모으고, 밤에는 몬스터로부터 살아남으세요!

## 🎨 기획

이 게임은 7살 아이의 아이디어에서 시작되었습니다.

## ✨ 주요 기능

### 생존 시스템
- **HP** - 체력 관리
- **배고픔** - 음식 섭취 필요
- **정신력** - 베이스캠프에서 회복

### 낮/밤 사이클
- 60초 주기로 낮과 밤이 전환
- 밤에는 더 강력한 몬스터 출현
- 정신력이 빠르게 감소

### 몬스터
| 주간 | 야간 |
|------|------|
| 🐍 뱀 | 🐛 거대 지네 |
| 🐒 원숭이 | 👤 광신도 |
| 🐻 곰 | 🐺 늑대 무리 |
| 🐜 불개미 | |
| 🧚 숲 요정 | |
| 🗿 고대 골렘 (50일+) | |

### 전투
- 5종 무기 (주먹 → 나뭇가지 → 돌도끼 → 창 → 횃불 → 강화 도끼)
- 콤보 시스템 (최대 5콤보, 20% 추가 데미지)

### 크래프팅
- 나무, 돌, 섬유 등 자원 수집
- 무기 및 도구 제작
- 회복 아이템 제작

### 이벤트
- 짙은 안개, 폭풍, 몬스터 습격
- 핏빛 달, 지진, 자원 풍요
- 보름달, 숲의 축복, 안개 속 그림자

### 보스
- 99일차에 **숲의 수호자** 등장
- 3단계 페이즈 전환
- HP 3000

## 🎮 조작법

### 데스크톱

| 키 | 동작 |
|----|------|
| WASD | 이동 |
| Shift | 달리기 |
| Space | 점프 |
| 마우스 | 시점/공격 |
| I | 인벤토리 |
| C | 크래프팅 |
| E | 자원 수집 |
| ESC | 일시정지 |

### 모바일

- 왼쪽 조이스틱으로 이동
- 오른쪽 터치로 시점 회전
- 화면 버튼으로 액션

## 📱 기능

- ✅ PWA 지원 (오프라인 플레이, 홈 화면 설치)
- ✅ 모바일 터치 지원
- ✅ 저장/불러오기
- ✅ 튜토리얼
- ✅ 업적 시스템 (12개)
- ✅ 다국어 지원 (한국어/영어)

## 🛠 기술 스택

- **Frontend:** React 18, Vite
- **3D:** Three.js, React Three Fiber, Drei
- **상태관리:** Zustand
- **오디오:** Howler.js
- **다국어:** i18next
- **배포:** Vercel

## 🚀 로컬 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 📁 프로젝트 구조

```
src/
├── components/
│   ├── game/       # 게임 오브젝트
│   ├── effects/    # 시각 효과
│   ├── lowpoly/    # 3D 모델
│   └── ui/         # UI 컴포넌트
├── stores/         # Zustand 상태
├── systems/        # 게임 시스템
├── data/           # 게임 데이터
├── i18n/           # 다국어
├── hooks/          # React hooks
└── utils/          # 유틸리티
```
